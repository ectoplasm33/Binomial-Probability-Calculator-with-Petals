# Binomial-Probability-Calculator-with-Petals

This Binomial Probability Calcualtor is desgined for general use with this version specifically including a petal conversion for htpps://florr.io players.

## How do I use it?
- To use the program down load the latest .zip file.
- Once downloaded, right click the file and click extract all.
- Open the extracted folder and run the executeable file.
- The executeable will trigger a SmartScreen alert; just click "More info" and "Run anyway".

## How do I use it? 
- Once the program has been run you will see a menu button in the top left corner of the window: open it.
- Once the side bar menu is open you can click the information icon above the text box to read about how it works and the controls.
